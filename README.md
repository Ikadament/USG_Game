# "Unfinished Square Game With Unnecessary Subtitle"

A 2D platformer puzzle game built with Unity.  
This project is a short game featuring simple movement, platforming mechanics, and puzzle interactions.

The game is available and playable directly in the browser on itch.io.

**Play here:**  
https://ikadam.itch.io/unfinishedsquaregame![Uploading image.png…]()


---

## About the Game

The core mechanic allows the player to change color, enabling them to pass through certain walls depending on their current color.
Colored boxes appear in each stage. They can be pushed through walls that match the opposite color of their own, just like the player.

---

## Features

- 2D character movement and jumping  
- Simple physics-based interactions  
- Puzzle elements and small challenges  
- Minimalist level design
- Interactable NPC with (humorous?) dialogue

---

## Controls

- **Left / Right** – Move  
- **Space** – Jump  
- **R** – Restart level  

---

## Technologies

- Unity (2D)
- C#
- WebGL build exported for browser play
